# GitHub 访问加速解决方案

## 推荐工具：Watt Toolkit (原 Steam++)

### 工具介绍
Watt Toolkit 是一款优秀的开源工具，专为 GitHub 加速而设计，支持以下功能：
- GitHub 网站访问加速
- GitHub 资源下载加速（包括代码仓库、Releases、ZIP 包等）
- 支持 Windows、macOS、Linux 等多平台
- 无需复杂配置，一键开启加速

### 手动安装步骤

1. **下载 Watt Toolkit**
   - 官方 GitHub 下载链接：[https://github.com/BeyondDimension/SteamTools/releases/latest](https://github.com/BeyondDimension/SteamTools/releases/latest)
   - 备用加速链接：[https://ghproxy.com/https://github.com/BeyondDimension/SteamTools/releases/latest](https://ghproxy.com/https://github.com/BeyondDimension/SteamTools/releases/latest)
   
   在下载页面中，找到 `SteamTools.Client.Windows.zip` 文件进行下载

2. **解压安装**
   - 下载完成后，将 ZIP 文件解压到任意位置
   - 运行解压目录中的 `SteamTools.exe` 程序

3. **使用 GitHub 加速功能**
   - 启动软件后，在左侧菜单中找到 "GitHub" 选项
   - 点击 "启用加速" 按钮
   - 加速功能启用后，您的浏览器访问 GitHub 将会自动加速

## 其他加速方案

### 1. 浏览器扩展方式

#### Fast-GitHub 扩展
- 支持 Chrome、Edge、Safari 浏览器
- 主要功能：GitHub 仓库下载加速、Release 附件加速
- 可在各浏览器的扩展商店搜索 "Fast-GitHub" 安装

### 2. 代理网站方式

以下是一些常用的 GitHub 代理网站：

1. **GitHub Proxy**：[https://ghproxy.com/](https://ghproxy.com/)
   使用方法：在 GitHub 链接前加上 `https://ghproxy.com/`
   例如：`https://ghproxy.com/https://github.com/user/repo.git`

2. **FastGit**：[https://doc.fastgit.org/](https://doc.fastgit.org/)
   提供多种使用方式，包括 Git 克隆加速

3. **GitClone**：[https://gitclone.com/](https://gitclone.com/)
   专注于 Git 仓库加速克隆

### 3. Hosts 配置方式

如果您熟悉 Hosts 文件配置，可以尝试更新 GitHub 相关域名的 IP 解析：

1. 访问 [https://github.com.ipaddress.com/](https://github.com.ipaddress.com/)
2. 查找并记录 GitHub 相关域名的最新 IP 地址
3. 编辑系统 Hosts 文件（路径：`C:\Windows\System32\drivers\etc\hosts`）
4. 添加 GitHub 域名的 IP 映射

## 注意事项

1. 建议使用 Watt Toolkit，这是目前最稳定可靠的 GitHub 加速工具
2. 代理工具的可用性可能会随时间变化，请及时更新到最新版本
3. 如遇工具无法使用，请尝试其他加速方案
